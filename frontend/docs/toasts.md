# Toasts & Notifications

**File**: `src/components/Toaster.jsx`

## Library

**Sonner** - React toast notification library

## Usage

Toasts are automatically triggered from API service functions:
- Success toasts for successful operations
- Error toasts for failures (with user-friendly error messages)

## Configuration

- **Position**: Centered at top of screen
- **Auto-dismiss**: Automatically dismisses after a few seconds
- **Manual Dismiss**: Users can manually dismiss
